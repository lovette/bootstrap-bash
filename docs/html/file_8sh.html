Copy directory and contents.

Parameters
srcpathSource directory path

ownerNew file ownership; set to empty string to preserve

Copy a file.

Parameters
srcpathSource file path
destpathDestination file path
ownerNew file ownership; set to empty string to preserve (optional)
permsNew file permissions; man chmod for allowed formats; set to 0 to preserve (optional)
overwriteOverwrite mode: 0=never, 1=always (default), 2=if newer or file size has changed (optional)
Note

Returns
Zero if successful, calls bootstrap_die otherwise
